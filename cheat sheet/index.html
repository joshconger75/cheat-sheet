HTML TABLE REFERENCE

HTML headings

Tag Name Description
<h1> h1 element h1 represents the top level heading
<h2> h2 element h2 represents the subheading
<h3> h3 element h3 represents the sub-subheading
<h4> h4 element h4 represents the sub-sub-subheading
<h5> h5 element h5 represents the sub-sub-sub-subheading
<h6> h6 element h6 represents the last sub-sub-sub-sub-subheading

Other text fundamentals

Tag Name Description
<p> Paragraph This denotes a paragraph mark up
<ul> Unordered Lists This makes lists without numbering
<ol> Ordered Lists This makes numbered lists
<li> List Item This adds items to ordered and unordered lists
<em> Emphasis Browsers style this as italic by default
<strong> Strong Importance Browsers style this as bold by default
<span> Span Span is an inline element used for making CSS changes in text or with in block level code
